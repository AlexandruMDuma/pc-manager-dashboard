## Example: Design my page

### Prompt

Design my page: a software catalog landing page with:
- A hero section explaining the value
- Search input
- Filter chips
- Cards grid with “flip to reveal details”
- Responsive layout and keyboard accessibility

### What the skill should do

1. Detect stack (React / Bootstrap / vanilla) + token setup.
2. Recommend MDS-first approach.
3. Propose:
   - Page structure (sections)
   - Recommended MDS components (if React)
   - Token-driven CSS guidance (if vanilla)
4. Include accessibility + responsiveness notes.

### Expected output shape (abbreviated)

- Detected stack summary
- Concrete steps (files to add/change)
- Components + tokens list
- Migration notes (if needed)
- Accessibility & responsiveness
